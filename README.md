# My Gift Card

### Unique system for accessing the gift certificate received

API:<br />
https://github.com/gabriel-cardoso-oliveira/my-gift-card-server<br />

## Quick start

Install:

```
git clone https://github.com/gabriel-cardoso-oliveira/my-gift-card-client
```
```
cd my-gift-card-client/
```
```
yarn install
```

Execute:

```
yarn start
```
